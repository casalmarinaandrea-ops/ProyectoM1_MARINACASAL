# ProyectoM1_MARINACASAL
# Generador de Paletas de Colores

Aplicación web estática e interactiva que permite generar paletas de colores aleatorias a partir de un botón principal.

## Funcionalidades

* Generación de paletas de colores aleatorias.
* Selección del tamaño de la paleta: 6, 8 o 9 colores.
* Visualización de cada color junto a su código HEX.
* Generación de colores en formato HEX.
* Botón principal “Generar paleta” operativo.
* Renderizado dinámico según el tamaño seleccionado.
* Microfeedback visible mediante tooltip, toast u otro mensaje de confirmación.
* Diseño funcional para desktop.

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript

## Accesibilidad

El proyecto incluye HTML semántico, labels asociados a los controles, contraste suficiente y foco visible para mejorar la navegación y la experiencia de uso.

## Objetivo del proyecto

El objetivo de esta aplicación es practicar la manipulación del DOM, la generación dinámica de contenido, el uso de eventos en JavaScript y la creación de una interfaz clara, accesible e interactiva.

## Como utilizar la web:
1. Seleccionar la cantidad de colores (6, 8 o 9).
2. Presionar el botón "Generar paleta".
3. Visualizar los colores generados junto a sus códigos HEX.
4. Generar nuevas combinaciones tantas veces como se desee.

## Caracteristicas implementadas:
- Generación aleatoria de colores.
- Formato HEX.
- Renderizado dinámico de la paleta.
- Microfeedback visual para el usuario.
- HTML semántico.
- Accesibilidad básica (labels, contraste y foco visible).
- Diseño responsive para desktop.
