# ProyectoM1_MARINACASAL
# Generador de Paletas de Colores

Aplicación web estática e interactiva que permite generar paletas de colores aleatorias utilizando HSL y visualizar sus equivalentes en HEX.

## Demo desplegada

https://casalmarinaandrea-ops.github.io/ProyectoM1_MARINACASAL/

## Funcionalidades

* Generación de paletas de colores aleatorias.
* Selección del tamaño de la paleta: 6, 8 o 9 colores.
* Generación de colores utilizando el modelo HSL.
* Conversión automática de los colores a formato HEX.
* Visualización de cada color junto a sus códigos HEX y HSL.
* Botón principal “Generar paleta” operativo.
* Renderizado dinámico según el tamaño seleccionado.
* Microfeedback visual mediante mensajes toast.
* Copia del código HEX y HSL al portapapeles al hacer clic sobre un color.
* Bloqueo y desbloqueo de colores para mantenerlos al regenerar la paleta.
* Guardado de paletas utilizando localStorage.
* Animaciones sutiles para mejorar la experiencia de usuario.
* Diseño funcional para desktop.

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript

## Accesibilidad

El proyecto incluye HTML semántico, labels asociados a los controles, contraste suficiente y foco visible para mejorar la navegación y la experiencia de uso.

## Objetivo del proyecto

El objetivo de esta aplicación es practicar la manipulación del DOM, la generación dinámica de contenido, el uso de eventos en JavaScript y la creación de una interfaz clara, accesible e interactiva.

## Como utilizar la web:
1. Seleccionar la cantidad de colores (6, 8 o 9).
2. Presionar el botón “Generar paleta”.
3. Visualizar los colores generados junto a sus códigos HEX y HSL.
4. Hacer clic sobre un color para copiar su código HEX o HSL o ambos al portapapeles.
5. Utilizar el botón de candado para bloquear o desbloquear colores.
6. Presionar “Guardar paleta” para almacenar la paleta actual.
7. Generar nuevas combinaciones tantas veces como se desee.
